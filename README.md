# 📊 Crime Trends Dashboard | Power BI

This project analyzes crime patterns in Los Angeles between **2020 and 2024**, using Power BI, DAX, and storytelling techniques to identify trends, peak hours, and crime distribution.

---

## 🎯 Project Objective
The goal of this dashboard is to analyze:
- crime evolution over time
- year-over-year variation
- top crime categories
- peak crime hours
- most affected regions
- victim demographic patterns

---

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
- Data Cleaning
- CSV Dataset
- Data Storytelling

---

## 📈 Key Insights
- Crime increased consistently from 2020 to 2022, peaking in 2022
- 2023 remained stable with only a slight decline
- 2024 shows a sharp drop, possibly due to partial data
- Vehicle theft remained the top crime category
- Crime peaks between **6 PM and 10 PM**
- Higher activity occurs on **Fridays and Saturdays**
